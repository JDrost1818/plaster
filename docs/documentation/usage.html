Usage

Plaster is a command line tool which provides a multitude of options in order to change behavior. The general syntax of the tool is as follows:

The basic syntax can be broken down into four parts:

  • Mode
  • Scope
  • Class Information
  • Configuration

Think of each as a key which unlocks different options down the tree. In better terms, different modes have different scopes, which in turn require different class information. Therefore, each mode has slightly different options and to best understand it, one should look at the documentation on the different modes:

  • Generate
  • Delete
  • Modify
